## Categories

| Category | Description |
|----------|-------------|
| **Skills** | PDF, Office, browser, email, news, monitoring |
| **Agents** | Pre-built agent configurations and personalities |
| **Prompts** | System prompts and templates |
| **MCP Servers** | Model Context Protocol integrations |
| **Themes** | Console UI themes and customizations |
| **Workflows** | Multi-step automation recipes |

## Publishing

```bash
prowlr marketplace publish ./my-skill/
```

See [templates/](templates/) for starter templates.
