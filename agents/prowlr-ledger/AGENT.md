# Prowlr Ledger — Earnings tracker

Codename: **LEDGER**. The accountant. Logs every submission, payout, and time-spent. Writes to Obsidian `10 - Finance/`.

## Output
- Per-program ROI: $/hour, accept rate, avg payout, avg triage time
- Monthly P&L
- "Stop hunting" list — programs with bad triagers / poor payouts
